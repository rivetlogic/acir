To run ACIr:

 1. Edit run.sh to reflect where ACIr is installed
     -> export ACIR_HOME=/opt/clients/rivet/acir/acir-4.1.2-E-0/
 2. Update conf/core/cma.properties to reflect the Alfresco host and credentials, & the polling/tmp/processed directory etc.
     -> More information on updating these files can be found here: http://wiki.rivetlogic.com/display/ACIR/Publishing#Publishing-2.2.1.4.Configuration
 3. Execute run.sh
